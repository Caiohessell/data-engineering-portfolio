# 📚 Estudos de SQL

Esta pasta reúne meus estudos de SQL utilizando **PostgreSQL**, organizados por tópicos para documentar minha evolução na linguagem. Cada diretório aborda um conceito específico, desde consultas básicas até técnicas mais avançadas.

Durante os estudos, utilizei como base um **dataset de E-commerce**, composto por tabelas relacionadas que simulam um ambiente de vendas online. Esse conjunto de dados permitiu praticar consultas em um cenário próximo ao encontrado em aplicações reais, envolvendo clientes, produtos, pedidos, pagamentos, avaliações, fornecedores e entregas.

## 📂 Organização dos estudos

| Pasta                  | Conteúdo                                                                         |
| ---------------------- | -------------------------------------------------------------------------------- |
| `01_select`            | Consultas básicas com `SELECT`.                                                  |
| `02_where`             | Filtragem de dados utilizando `WHERE`.                                           |
| `03_order_by`          | Ordenação de registros com `ORDER BY`.                                           |
| `04_funcoes_agregacao` | Utilização de `COUNT`, `SUM`, `AVG`, `MIN` e `MAX`.                              |
| `05_group_by_having`   | Agrupamento de dados e filtragem de grupos.                                      |
| `06_joins`             | Relacionamentos entre tabelas utilizando JOINs simples e múltiplos.              |
| `07_subqueries`        | Subconsultas, tabelas derivadas e consultas correlacionadas.                     |
| `08_ctes`              | Common Table Expressions (CTEs) para consultas mais organizadas e reutilizáveis. |

> Novos tópicos serão adicionados conforme avanço nos estudos.

## 🗃️ Dataset utilizado

Os exercícios foram desenvolvidos sobre um banco de dados de **E-commerce**, modelado para representar operações comuns de uma loja virtual.

Principais tabelas utilizadas:

- Clientes
- Pedidos
- Itens do Pedido
- Produtos
- Categorias
- Fornecedores
- Funcionários
- Pagamentos
- Entregas
- Avaliações

A utilização desse dataset possibilitou praticar consultas envolvendo relacionamentos entre tabelas, agregações, filtros, subconsultas e análises de dados em um contexto de negócio.
